CSS ( Cascading Style Sheets )

adalah bahasa yang digunakan untuk mengatur tampilan dan desain halaman website. CSS bekerja bersama HTML untuk membuat tampilan website menjadi lebih menarik, rapi, dan mudah digunakan. Jika HTML berfungsi sebagai struktur atau kerangka, maka CSS berfungsi sebagai penghias atau desainnya, seperti mengatur warna, ukuran, posisi, dan tata letak elemen.
