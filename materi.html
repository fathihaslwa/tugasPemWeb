Cascading Style Sheets
CSS (Cascading Style Sheets) adalah bahasa gaya bahasa (style sheet language) yang digunakan untuk mengatur tampilan visual elemen yang ditulis dalam bahasa markup seperti HTML. CSS berfungsi memisahkan konten (HTML) dari desain, memungkinkan pengaturan tata letak, warna, font, dan responsivitas situs agar lebih menarik.
PENTINGNYA BELAJAR CSS
  • Membuat website menarik : mengatur warna, font, layout, dan animasi.
  • Memisahkan konten & desain : HTML untuk struktur, CSS untuk tampilan.
  • Responsif : tampilan optimal di berbagai perangkat (desktop, mobile).
